# Pondermatic

an AI toolchain using Crystal Language

don't read farther without basic Crystal Language comprehension

install Rakefile and run

on a Debian just run

$ sh install.sh

if you have no Debian, run one in a virtual box.

Now get started

$ rake  #  to build and test

or

$ rake spec  #  to test

or

$ rake sound[frog]  #  to simulate success
